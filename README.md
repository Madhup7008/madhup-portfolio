
# Madhup Kumar Yadav - Portfolio

This is a high-performance personal portfolio website built with:
- **Next.js 15** (App Router)
- **Tailwind CSS 4**
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Customization

-   **Content**: Edit `src/data/portfolio.ts` to update your bio, skills, and projects.
-   **Styling**: Modify `src/app/globals.css` for theme colors.
-   **Components**: Check `src/components/` to tweak layouts.
